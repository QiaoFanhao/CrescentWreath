## 5) 当前可直接落地到实现清单的“规则点索引”

> 下面这些点可以直接作为后续任务的验收项（DoD）引用。

1. 回合阶段固定为：开始 / 行动 / 召唤 / 结束。  
2. 伤害类型至少区分：体术 / 咒术 / 直接；直接伤害默认不可防御。  
3. 防御是正式动作（不是单纯减伤数字），且依赖类型匹配。  
4. 响应技不消耗技能值，并受“同一时点不可重复响应”约束。  
5. 异变存在“当前异变对象 + 解决条件 + 奖励 + 时点限制”。  
6. 召唤获得的宝具默认进入弃牌区（而非直接入手）。  
7. FAQ 已确认多处“触发时点优先于结果状态”的裁定原则。
