## 5) 可直接落地的规则点索引（更新）

1. 回合阶段固定为：开始 / 行动 / 召唤 / 结束。
2. 伤害类型至少区分：体术 / 咒术 / 直接；直接伤害默认不可防御。
3. 灵符必须区分 `sigilPreview` 与 `lockedSigil`，并在召唤阶段开始锁定。
4. 放逐语义需区分“永久 `GapZone` 锁定”与“可再引用放逐对象”。
5. 响应技不消耗技能值，且同一时点不可重复响应。
6. 冲突裁定遵循“不 能 > 能”；可选效果不能当作强制效果。
7. 叠放语义挂到 `OverlayContainer`，不做散落特判。
8. 盖放语义挂到 `CharacterSetAside`，并支持可序列化恢复。
9. 异变流程显式维护 `currentAnomalyState`，且一回合最多解决一次。
10. 规则裁定由服务器权威执行，客户端只做输入与展示。

3. 本轮冲突已按“正式规则书 + FAQ 优先”完成裁定，可进入定义层冻结。
